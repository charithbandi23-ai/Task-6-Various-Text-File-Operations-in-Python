6.3 Library Management


Scenario: A library keeps a file books.txt with book titles, some titles repeated. Write a program to create a new file with only unique book titles.

Algorithm:

1. Open books.txt in read mode.

2. Read all book titles into a set to remove duplicates.

3. Close the file.

4. Open unique-books.txt in write mode.

5. Write all unique book titles into the file.

6. Close the file.

7. Display unique book list created.

Sample File: books.txt
Python programming
Data structures
machine learning
Python programming

Code:
with open("books.txt", "r") as f:
    books = set(f.readlines())
with open("unique-books.txt", "w") as f:
    for book in books:
        f.write(book)
print("Unique book list created.")
